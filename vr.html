VR Environments

With Unreal Engine we builded environments and we learned how to work with the technique Projection Mapping by using Styrofoam. Spatial Augmented Reality (aka Projection Mapping) uses everyday video projectors, but instead of projecting on a flat screen (e.g. to display a PowerPoint), light is mapped onto any surface, turning common objects of any 3D shape into interactive displays. It can be projected on really small or big surfaces and even can be interactive. The Unreal Engine Assignment consists of two parts. In the first part we learned to use materials, textures and movement to create a turning banana on a turntable in 3D. In the seconde one we created a small game where you can collect bananas as a red cube character.


Unreal Engine Assigment

Banana turntable

I learned how to style objects by creating materials and adding them to a specific object, in this case a banana. Then I had to create a blueprint for my banana. With a blueprint you can create a certain behavior for objects without having to code, such as galloping horse or a driving car. By adding a local rotation, an object can rotate. By linking this blueprint to the banana, the banana can rotate:

Eat bananas

In this excercise I learned even more about blueprints in Unreal Engine. I've added bananas which the red cube can eat. I made this happening by using the 'DestroyActor', 'AddLocalRotation', 'Event Trick', 'Rotation Speed', 'Multiplication' and 'On Component Begin Overlap'. The red cube can not go through the walls.


Spatial Augmented Reality

We learned to projection map images on Styrofoam using a projection mapping software called MadMapper. During the workshop the projectors were already connected to the PCs, so we could get started right away. Because there were only two projectors, we worked in groups with five people. To experiment with images and videos, we used a polystyrene ball on which we projected an image of the earth. Then we adjusted the opacity that was influenced by the height of the decibels of the sounds we played. We used a video of a rotating earth so it looked like it came alive and the earth was spinning in real life.


After the earth we wanted to work with faces. We first used a mannequin, where we projected a mask over the eyes. We also wanted to see how it will look like if we use another face on someone's face. Both examples you can see in the videos.
