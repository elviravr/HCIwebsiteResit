Computer Vision

During the ‘computer vision’ workshop I started working with Processing again. I worked with it once before during the "Arduino" workshop. Processing is an open source programming language and environment for people who want to program interactive images and animations.


Assignment 1: Figuring

In the first assignment I had to make three figures, each with a different location, different colors, different widths and heights, and some with a border and some without. This had to be made with the Processing software. I used a void setup function to activate this.


Assignment 2: Making a body

In this assignment I had to write a code in which there were at least four different figures. By making a circle, a rectangle, triangles and lines, I created a body. I also made this code in Processing. To create these figures, I also had to use a void setup function.


Assignment CV1: Green circle on face

In this assignment I needed to change a rectangle to a circle and by using facial recognition (from the openCV library) I could set the circle on a face via the webcam.


Assignment CV2: Rudolph nose

The purpose of this assignment was to have a red nose on every person who appeared on screen. For this I had to change the code. The opencv.loadCascade had to be adjusted to recognize the nose. The fill of the circle needed to be red, so that the circle around the nose turned red. To make this command a success, a void setup was used, void draw and a void capture Event.


Bonus Assigment

A bonus assignment. Via the webcam and the code it recognizes lips and by using the correct code you can put a a image on your lips.


Assignment CV3: Color tracking

In this assignment I had to build something creative. For inspiration I looked at the examples in the folder of Processing. I used the "OpenCVMultipleColorTrackingLiveCam". With this you can press the buttons 1, 2, 3 and 4. When I runned this Color Tracking program, a camera window opened up along with the object detected by the webcam or a laptop camera. Colour detection is indicated by rectangular boxes around the objects along with the names of the colours: red, blue and yellow. Unfortnately I didn't make a screenvideo of it.


Assignment AR

For the AR assignment I had to download another library. The downloaded library would ensure that the webcam with processing recognizes certain "markings" and places in front of it. What was needed for this was the code of Hiro. When you placed it in front of your camera, a blue cube appeared on this spot.
