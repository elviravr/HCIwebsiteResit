Arduino

An Arduino is open-source electronic prototyping platform enabling users to create interactive electronic objects. We learned the basics of prototyping with an Arduino with multiple sensors, lights, resistors, wires and other small electronics. To see what I've learned during the workshops, scroll through the exercises.


3c: Blink multiple leds

The first exercise was about how to make a LED blink. Because this was not exciting enough, we extended the circuit with a second LED. The 2 LEDs needed to blink at the same time but one of the LEDs needed to blink twice as fast as the other. I used jumping wires, resistors and the LED's. Below you can find the results.



4c: Fade LED - Analog output

We can do more than just turn a LED on and off. We can also control the brightness of LEDs. With the analogWrite() function we can make the LEDs fade in and out. In this example the objective was to make one LED fade in while the one LED slowly fades out. The minimum of the brightness is 0 and the maximum 255. I used the same materials for this exercise as the previous one. The circuit is also almost exactly the same, the only difference is the placing of the jumping wires and the code. See the result below.



6c: Controlling a LED with a potmeter

Another possibility to control the LEDs is with a potentiometer. This is a 'turning button' and by connecting it to the Arduino we can read its value. Then it's possible to use this value to control the LED. If I turned the potentiometer, then one LED should fade in and the other one fade out. In the code are some changes again, with the use of Serial.print and a map attribute and sensorValue to read the values.



7: Control the LED with light

A Light-Dependent Resistor or LDR is a sensitive resistor that has its resistance depending on the amount of light it's relieving. Additionaly, I used again LEDs, resistors and jumping wires. By mapping the output of the LDR sensor, we can control the LED. To see how big the difference is between the dark and light, we opened the Serial Monitor. First the sensorValue must be read which passes the value and shows it in the Serial Monitor.

9b: Arduino and processing

Another program called Processing is introduced to get the result for this exercise. I also used LDRs, resistors, LEDs and jumping wires. The result was a figure in Processing that changes the size when the amount of light changes through the LDRs. The more light, the bigger the figure and vice versa.



10f: Buttons

I created two interaction inputs by putting two buttons in my breadboard and connected them to Arduino. Besides those materials, I also used a LED, resistors and jumping wires. The task in this assignment was to only make the LED light up when only one button is pressed, but not when both buttons are pressed. The "if" statement tells when the LED needs to go on. With the "else if" statement the LED needs to be switched off when the "if" conditions aren't met.



11b: Servometer

Servomotors are motors that turn to a precise position that are mostly used in remote controlled vehicles. Most servos can not rotate completely, usually they have a reach of 180 degreedes. We needed to change the code and let the servo dance. I used the servomotor and jumping wires. To complete this task, I setted different positions and used a for loop that iterated through positions. When I uploaded the code, the servo did his dance.



12b: Controlling the servo with inputs

To practice more with the servometer, we combined it with buttons which controlled the servo. If you press the button, the servomotor needed to do something. One button is used to rotate the servo clockwise and the other button is to rotate the servo counterclockwise. 'Pos' in the code means the position of the servometer. The -- and ++ behind it, stands for the side where the servo needs to go to. Because there are more scenario's, I used an 'if' statement.



13b: Sound

Done with the servodances, now let's make music! With the buzzer connected to the Arduino and checking the frequencies, I was able to create tunes. I used a Piezo speaker, jumping wires and a resistor for this exercise. I wanted to create a nice little melody and here in the video you can find the result.
